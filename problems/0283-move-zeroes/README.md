# #0283 - Move Zeroes

- Difficulty: Easy
- Solved: 2026-05-19
- LeetCode: https://leetcode.com/problems/move-zeroes/
- Topics: Array, Two Pointers, In-place

## Question

TR: Sifirlari sona tasirken sifir olmayan elemanlarin sirasini koru.

EN: Move zeroes to the end while preserving the order of non-zero values.

## Idea

TR: Once sifir olmayanlari basa yaz, sonra kalan yerleri sifirla.

EN: First compact non-zero values, then fill the remaining positions with zero.
