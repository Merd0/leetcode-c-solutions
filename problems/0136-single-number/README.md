# #0136 - Single Number

- Difficulty: Easy
- Solved: 2026-05-19
- LeetCode: https://leetcode.com/problems/single-number/
- Topics: Array, Bit Manipulation, XOR

## Question

TR: Digerleri iki kez gecen dizide sadece bir kez gecen sayiyi bul.

EN: Find the number that appears once when every other number appears twice.

## Idea

TR: XOR islemi ayni sayilari sifirlar, geriye tek kalan sayi kalir.

EN: XOR cancels equal pairs and leaves the single value.
